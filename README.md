# My Portfolio

This is a modern portfolio website built with React, Vite, Tailwind CSS.

## Features
- Responsive design
- Hero section
- AI Assistant component
- Portfolio data driven

## Tech Stack
- React
- Vite
- Tailwind CSS
- PostCSS

## Getting Started

### Prerequisites
- Node.js

### Installation
1. Clone the repo
```
git clone https://github.com/Rajsonipro/-My-portfolio.git
```
2. Install dependencies
```
npm install
```
3. Run dev server
```
npm run dev
```

## Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
